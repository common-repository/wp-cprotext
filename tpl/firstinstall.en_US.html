WELCOME TO WP-CPROTEXT

Thanks for installing the CPROTEXT.COM online service WordPress plugin.

The CPROTEXT.COM online service allows you to protect texts and images published on the web to avoid non authorized copies. This plugin integrates that service within the WordPress publication process in an unobtrusive way. It requires to have registered for a free account that includes 5 free protected contents on the CPROTEXT.COM website

FIRST STEPS

  1. Check that you have registered for a free account including 5 free protected contents on CPROTEXT.COM.
  2. Click on the "API" tab of your CPROTEXT.COM account and generate an authentication token for this WordPress Instance.
  3. Click on the "CPROTEXT" tab in the WordPress "Settings" menu.
  4. Copy/paste your authentication token in the approriate text box in the WordPress CPROTEXT Settings.
  5. Import your CPROTEXT.COM account settings.
  6. If not already available in the fonts list, upload the font that should be used to display your texts on this WordPress blog.
  7. Select this font as the default font.
  8. Save these modifications.

IMPORTANT INFORMATION

For security concerns, the authentication token available in your CPROTEXT.COM account must never be shared with a third party.
This plugin will never ask you for your account credentials, but only for the authentication token which is specific to this WordPress instance.

TEXT PROTECTION

After you wrote your text in the editor panel, you can ask for its protection by activating and filling the CPROTEXT frame before clicking the WordPress "Publish" button.

IMAGE PROTECTION

For each image you wish to protect, go in the media library, select the image and check the "Protected" checkbox.

If you have any questions related to the CPROTEXT.COM online servive or to this plugin, please first check the F.A.Q. on the CPROTEXT.COM website.
