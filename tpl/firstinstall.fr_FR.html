BIENVENUE DANS WP-CPROTEXT

Merci d'avoir installé l'extension WordPress du service en ligne CPROTEXT.COM.

Le service en ligne CPROTEXT.COM permet de protéger les textes et les images publiés sur le web afin d'éviter les copies non autorisées. Cette extension intègre ce service dans le processus de publication de votre blog WordPress de manière quasi transparente. Elle nécessite d'avoir préablablement créé gratuitement un compte incluant 5 contenus protégés sur le site CPROTEXT.COM.

PREMIERS PAS

  1. Assurez-vous d'avoir ouvert un compte gratuit incluant 5 contenus gratuits sur CPROTEXT.COM.
  2. Cliquez sur l'onglet « API »" de votre compt CPROTEXT.COM et générez un jeton d'identification pour ce blog WordPress.
  3. Cliquez sur l'onglet « CPROTEXT » du menu « Réglages » de WordPress.
  4. Copier/coller votre jeton d'identification dans la boite de texte des Préférences CPROTEXT de votre blog WordPress.
  5. Importez les préférences de votre compte CPROTEXT.COM.
  6. Si elle n'est pas déjà présente dans la liste des polices disponibles, ajoutez la police de caractères qui doit être utilisée pour l'affichage de vos textes dans ce blog WordPress.
  7. Selectionnez cette police de caractères comme la police par défaut.
  8. Enregistrez ces modifications.

REMARQUE IMPORTANTE

Pour des questions de sécurité, le jeton d'identification disponible dans votre compte CPROTEXT.COM ne doit jamais être partagé avec un tiers.
Cette extension ne vous demandera jamais les identifiants de votre compte, mais uniquement le jeton d'identification spécifique à ce blog WordPress.

PROTECTION DE TEXTE

Après avoir rédigé votre texte dans la page d'édition de WordPress, vous pouvez demander à le protéger en activant et remplissant le cadre CPROTEXT avant de cliquer sur le bouton « Publier ».

PROTECTION D'IMAGE

Pour chaque image que vous souhaitez protéger, rendez-vous dans la bibliothèque de médias, sélectionnez une image et cochez la case "Protégée"

Pour toutes questions relatives au service en ligne CPROTEXT.COM ou à cette extension, consultez préablablement la F.A.Q. sur le site de CPROTEXT.COM
